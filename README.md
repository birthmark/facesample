# facesample
https://blog.csdn.net/cuiran/article/details/85276384

本项目主要基于vlc来播放流媒体视频
可以入QQ群参与讨论:735436509

主要包含以下内容

- 1、使用已经编译的libvlc来播放流媒体视频
- 2、使用MTCNN进行人脸识别并标记人脸
- 3、保存标记的人脸图片
- 4、使用FACENET进行人脸比对
- 未完待续...
